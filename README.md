# Mood Moon

A professional B2B website for a lighting design company based in Minsk, Belarus. The site showcases lighting design services for public spaces and interiors.

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, 3D transforms)
- JavaScript (vanilla)
- Google Fonts (Inter)

No build tools or external dependencies required.

## Project Structure

```
Mood-Moon/
├── index.html          # Main single-page website
├── style.css           # Styling (BEM methodology)
├── scripts/
│   └── script.js       # JavaScript functionality
└── img/                # Image assets
    ├── design/         # Design process images
    ├── equip/          # Equipment brand logos
    ├── icons/          # Social media icons
    └── lightning/      # Lighting plan visuals
```

## Features

- **Interactive flip cards** with 3D hover animations
- **Responsive grid layouts** for galleries and equipment sections
- **Modern UI** with smooth transitions and hover effects
- **Clean design** with beige/dark color scheme

## Sections

1. Header with navigation
2. Hero section
3. Project gallery
4. Design process (6 steps with flip cards)
5. Pricing information
6. Equipment partners (12 brands)
7. Lighting scenarios
8. Service features
9. Footer with contacts

## Getting Started

Simply open `index.html` in a browser. No build process required.

```bash
# Or serve locally
npx serve .
```
